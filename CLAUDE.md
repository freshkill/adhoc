# Repository Guidance

## Git Workflow Preference

- **Push changes directly to `main`** by default. Do not create a separate feature branch unless explicitly instructed to.
- **Do not open pull requests** unless explicitly instructed to. Commit and push straight to `main`.
- This preference applies to this repository. If a session is configured with its own designated branch, follow the user's explicit instruction here (push to `main`) when the user confirms.
